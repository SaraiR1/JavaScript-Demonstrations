How I Learned Web Development

I started learning web development 10 months ago. In the summer, my first programming language class was Python. After making many projects with Python, learning C++ and JavaScript was easier. In the Fall Term, I learned how to make web pages using HTML and CSS. I made multiple web pages in various designs! It is amazing how much you can do with CSS! You can change the layout of the page, add transitions, and add different animations. JavaScript is a scripting language that I understood well. I used JavaScript on my web pages to validate forms, display alerts, generate HTML markup tags, and prompt the user for information. I have created web pages that included forms and was able to validate the user's input by creating regular expressions, specifying a range for numeric inputs, and making certain fields required. I anticipate creating and designing more complex websites in the future!

When Did I Start?

5 months ago, I started making websites for clients. After creating many different web pages and using various CSS features, designing web pages has become an enjoyable task for me. Equally important, I can make web pages more interactive with JavaScript since I have been using JavaScript diligently in the past 5 months. I am confident in my skills and hope to grow my clientele!
